Register File

32 General purpose registers of 32 bits.

Description

Reads are asynchronous. Writes happen on rising edge. Register 0 is always 0 and cannot be written to.

Ports

clkClock signal
resetReset signal
rreg1Read data 1 from this register
rreg2Read data 2 from this register
rdata1Read data 1 output
rdata2Read data 2 output
regwriteOn 1 write to register
wregWrite to this register
wdataWrite this data
